# src/icons/

Place your PWA icons here (used by Android when installing as a home screen app):

| File          | Size     |
|---------------|----------|
| icon-192.png  | 192×192  |
| icon-512.png  | 512×512  |

These are referenced in `src/manifest.json`.

Generate them from your master icon using any image editor or:
https://realfavicongenerator.net
