# Changelog

All notable changes to LED Badge S1144 are documented here.

## [1.0.0] — Initial Release

### Added
- Full LED badge controller UI (Model S1144, 11×44 matrix)
- Indic script support (Gujarati, Devanagari, transliteration)
- Scroll modes: Left, Right, Up, Down, Fixed, Snow, and more
- Multi-message slots (up to 8)
- Brightness control
- Emoji picker
- Message history with resend
- Live LED dot-matrix preview with animation
- **Desktop** — Electron wrapper with WebHID auto-permission
- **Android** — PWA with WebUSB/OTG support via Chrome
- **CI/CD** — GitHub Actions builds Windows/macOS/Linux and deploys PWA to GitHub Pages
